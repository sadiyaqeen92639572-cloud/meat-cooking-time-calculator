# Meat Cooking Time Calculator

Free meat roasting-time calculators — enter meat type, cut, weight and doneness to get the exact
cooking time, oven temperature (°C & °F), internal target temperature and resting time.

**Live site:** _github.io Pages URL (add CNAME when the domain is bought)_

## Features

- Universal calculator (beef / lamb / pork / chicken / gammon) — homepage
- 4 meat hub calculators: roast beef, leg of lamb, roast pork, roast chicken
- ~25 cut-specific calculators (brisket, gammon, pork loin, shoulder of lamb, spatchcock chicken…)
- Weight-based programmatic pages (0.5–4 kg per meat, incl. non-round chicken weights)
- Informational feeder pages (chicken breast in the oven, internal temperatures, roast potatoes,
  roast dinner ingredients, US °F "roast a chicken at 350/325"…)
- Guidance ranges + "start hot then reduce" sear method, not a single rigid formula
- "Formulas & Method" block on every calculator (constants + sources) for SEO/GEO citability
- JSON-LD: WebApplication + FAQPage + HowTo + BreadcrumbList
- sitemap.xml auto-generated by `generate-pages.js`

## Build

```bash
node generate-pages.js   # emits all hub/cut/weight/informational folders + rebuilds sitemap.xml
```

The homepage `index.html` is hand-written; everything else is generated.

## Deploy

Push to `main` → GitHub Pages workflow (`.github/workflows/pages.yml`) publishes the repo root.
To use a custom domain: add a `CNAME` file with the domain and set `SITE_URL` in `generate-pages.js`
and the homepage canonical/OG tags, then re-run the generator.

## Monetization

Affiliate CTA order: (1) meat thermometer, (2) roasting equipment, (3) recipe boxes, (4) ads.
Replace the `AMAZON_TAG` / affiliate placeholder links before going live.

## Data sources & disclaimer

Cooking times are **guidance** compiled from standard UK roasting guides (BBC Good Food, Waitrose,
butchers' guides) applied deterministically — **not** a food-safety authority. Always verify
doneness with a meat thermometer to the stated internal temperature.
